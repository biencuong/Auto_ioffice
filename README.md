# Auto_iOffice Handover Bundle

This folder contains the iOffice AI Agent product, configuration, guidance, and handover materials.

## Quick start
- QwenPaw runtime: run `qwenpaw app`
- Agent setup: run `.\scripts\setup_qwenpaw_ioffice.ps1`
- Dashboard: run `cd ioffice-dashboard && node server.js`
- Core skill: `skills/ioffice-agent/`
- Memory: `memory/ioffice/`
- Integration guide: `docs/QWENPAW_INTEGRATION.md`
